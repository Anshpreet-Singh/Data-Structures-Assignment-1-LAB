# Assignment 1 – Data Structures and Algorithms

## Overview

This assignment introduces some fundamental concepts of **Data Structures and Algorithms (DSA)**. The goal of the assignment is to understand **Abstract Data Types (ADT)**, analyze the **time and space complexity** of common algorithms, and study different **algorithm design paradigms** such as recursion, divide and conquer, and dynamic programming.

The report explains the **Stack ADT**, evaluates the complexity of several algorithms, and reflects on how different algorithmic paradigms are applied in problems like **Binary Search, Fibonacci, and Tower of Hanoi**.

---

## Contents of the Assignment

### Part A – Stack ADT

This section explains the concept of an **Abstract Data Type (ADT)** and uses the **Stack** as an example.

Topics covered:

* Definition of Abstract Data Type
* Basic stack operations (push, pop, peek, isEmpty)
* Efficiency of stack operations
* Why stack operations generally run in **O(1) time**

---

### Part B & Part D – Complexity Analysis

This section analyzes the **time complexity** and **space complexity** of several algorithms.

Algorithms discussed:

* Factorial
* Fibonacci (Naive Recursive)
* Fibonacci (Memoized)
* Tower of Hanoi
* Binary Search

For each algorithm, the following aspects are analyzed:

* Time complexity
* Space complexity
* Algorithm design paradigm used

---

### Part E – Paradigm Reflection

This part discusses how different algorithms follow specific **algorithm design paradigms**.

Topics explained:

**1. Binary Search – Divide and Conquer**

* How the problem is divided into smaller halves
* Recurrence relation: T(n) = T(n/2) + O(1)
* Best, average, and worst case complexities

**2. Fibonacci with Memoization – Dynamic Programming**

* Problem of overlapping subproblems
* Concept of memoization
* Improvement from exponential complexity to linear complexity

**3. Tower of Hanoi – Recursion**

* Recursive breakdown of the problem
* Base case when n = 1
* Explanation of smaller subproblems
* Time complexity and number of moves

---

## Learning Outcomes

After completing this assignment, the following concepts should be understood:

* The purpose of **Abstract Data Types**
* Basic operations and efficiency of a **Stack**
* How to analyze **time complexity and space complexity**
* Understanding of **recursion**
* Understanding of **divide and conquer algorithms**
* Understanding of **dynamic programming and memoization**

## Conclusion

This assignment provides a foundational understanding of important algorithmic concepts used in computer science. By analyzing different algorithms and paradigms, it helps develop the ability to evaluate algorithm efficiency and choose appropriate approaches for solving computational problems.
